Ford Bikeshare Capstone

  • Python
  • Jupyter Notebook

The project consists of two components: The first is an analysis designed to uncover trends and relationships within the dataset to provide Ford with data-driven insights to help them adjust their approach, anticipate client needs, and meet demand more effectively. The second component is a report summarizing the key findings from the analysis.

Parsed and cleaned a Ford GoBike System dataset to ensure data integrity. Renamed columns appropriately and ensured values conformed to the expected schema. Manipulated the data to evaluate various aspects of the data, including user characteristics, ride distribution by day, time, and station. Utilized contingency tables and a chi square test to analyze the observed and expected frequencies of station usage across different days. Concluded the project with actionable insights, supported by compelling visualizations, elaborations on findings, and strategic recommendations.
